(constant) segmentStringIntoGCs

Creates a closure that caches the Intl.Segmenter instance for grapheme segmentation. This function is called only once when the script loads. The inner function (the actual segmenter) is then returned and used repeatedly, benefiting from the cached Intl.Segmenter object.
