Class: GCString

GCString(str)

GCString - A grapheme cluster aware string wrapper class. This class wraps JavaScript strings and provides methods that operate on user-perceived characters (grapheme clusters) rather than code units or code points. This ensures correct handling of emojis, combining characters, regional indicators, and other complex Unicode sequences.
